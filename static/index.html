UltraSinger

Full path to UltraSinger.py on this machine. Saved in the server settings file unless ULTRASINGER_PY is set in the environment (then it overrides and this field is read-only). Examples: /home/you/UltraSinger/UltraSinger.py (Linux) · D:\UltraSinger\UltraSinger.py (Windows).

Folders

Optional. Leave empty to use defaults under the app data directory (data/jobs and data/uploads, or data\jobs on Windows). Job folders are job_0001, … inside the output folder. Examples — output: /var/lib/ultrasinger/jobs · D:\UltraSinger\jobs; uploads: /tmp/us-uploads · C:\Temp\us-uploads.

Whisper

Passed to UltraSinger as --whisper_compute_type. Use int8 if you are low on VRAM.

About

UltraSinger WebUI is a browser companion for UltraSinger. It does not replace it: this app queues work, runs UltraSinger.py as a subprocess, and adds uploads, playlists, post-processing, ZIP downloads, and job cleanup around the upstream tool.

UltraSinger (upstream)

Powered by UltraSinger

UltraSinger is an open-source, AI-based pipeline that builds UltraStar karaoke text, MIDI, and note data from audio or video. It uses transcription (including Whisper), pitch detection, optional vocal separation, and related steps. Install it separately, then point this WebUI at UltraSinger.py under Settings.

Official project: github.com/rakuri255/UltraSinger — documentation, CLI options, GPU setup, and support channels are maintained there.

What you need

  • UltraSinger installed on the same machine as this app (path under Settings)
  • ffmpeg on PATH for audio conversion
  • yt-dlp on PATH for playlist expansion (optional)

Configuration

Server paths and folders are stored in webui_config.json under your configured data directory (see Settings). Whisper and export defaults for new jobs are stored in this browser.
