Settings

UltraSinger path and folders are stored in data/webui_config.json on the server. Whisper and export options below are stored in this browser.

UltraSinger

Full path to UltraSinger.py on this machine. Saved under data/webui_config.json unless ULTRASINGER_PY is set in the environment (then it overrides and this field is read-only). Examples: /home/you/UltraSinger/UltraSinger.py (Linux) · D:\UltraSinger\UltraSinger.py (Windows).

Folders

Optional. Leave empty to use defaults under the app data directory (data/jobs and data/uploads, or data\jobs on Windows). Job folders are job_0001, … inside the output folder. Examples — output: /var/lib/ultrasinger/jobs · D:\UltraSinger\jobs; uploads: /tmp/us-uploads · C:\Temp\us-uploads.

Whisper

Passed to UltraSinger as --whisper_compute_type. Use int8 if you are low on VRAM.
